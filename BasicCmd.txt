1. docker run nginx - Start a container
2. docker ps - List containers
3. docker ps -a - List of containers active or exited
4. docker stop container_name - Stop a container
5. docker rm container_name - Remove a container
6. docker images - List of images
7. docker rmi image_name - Remove an image_name
8. docker pull image_name - Pull / download an image
9. docker exec container_name cat /etc/hosts - Executes command on running container
10. docker run container -  Run or rather attaches to the container
11. docker run -d container - Detach from the container
12. docker attach cotainer_id - Attaches the container
13. docker run -d container_name - Runs the container in background
14. docker stop cotainer_id - Stops the running container
